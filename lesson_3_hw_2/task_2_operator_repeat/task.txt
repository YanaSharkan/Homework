2.1 Оператор (*) для рядків (str)

Оператор * ( повторення) належить до базових операторів для роботи з рядками. (Окрім нього також : + (конкатенація), [ ] (звертання до символу рядка s за індексом i), [:] ( витягування підрядка із заданої позиції до вказаної).
Для рядків, оператор * ( повторення) повертає рядок, повторений задане число разів. Ми можемо повторювати рядок скільки завгодно разів,  використовуючи оператор множення *.

Наприклад:
>>> 'apple' * 3
’appleappleapple’

Ми можемо використовувати тільки int як операнд для повторення рядків. Інакше це викличе наступну помилку:

>>> 'Python' * 'Python'
TypeError: can't multiply sequence by non-int of type 'str'

2.2 Оператор (*) для списків (list).
Список підтримує два оператори: + для конкатенації та * для повторення елементів.
Python list також підтримує оператор * (повторення) для створення нового списку з елементами, які повторюються вказану кількість разів.

>>> [5] * 3
[5, 5, 5]
